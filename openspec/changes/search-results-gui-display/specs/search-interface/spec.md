## MODIFIED Requirements

### Requirement: キーワード入力による検索実行
システムは、ユーザーが検索キーワードを入力し検索を実行した時に、サイトの検索結果を非同期で取得し、モーダル内で結果を表示しなければならない (SHALL)。

#### Scenario: キーワード入力による検索実行
- **WHEN** 入力フィールドにキーワードを入力し、Enterキーを押下または検索ボタンをクリックした時
- **THEN** アダプターを通じて生成された検索URLに対して `fetch` を実行し、レスポンスHTMLをパースして検索結果がモーダル内で表示されること

#### Scenario: 検索結果が0件の場合
- **WHEN** キーワード入力による検索実行の結果、検索結果が0件であった時
- **THEN** 結果が見つからないことを示すメッセージがモーダル内で表示されること
