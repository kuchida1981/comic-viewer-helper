# internal-tag-navigation (Delta)

## MODIFIED Requirements

### Requirement: タグベースの内部検索起動
システムは、メタデータモーダル内のタグがクリックされた際、外部サイトへ遷移するのではなく、そのタグを条件とした内部検索を実行しなければならない（SHALL）。検索開始時には以前の結果をクリアし、入力フォームの状態を適切に管理する必要がある。

#### Scenario: タグクリックによる検索実行時のクリーンアップ
- **WHEN** ユーザーがメタデータモーダル内のタグチップをクリックした時
- **THEN** 検索モーダルが表示される前に `searchResults` が `null` にクリアされる
- **AND** キーワード入力フォームが空の状態で表示される
