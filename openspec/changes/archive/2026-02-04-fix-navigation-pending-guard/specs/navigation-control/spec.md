## ADDED Requirements

### Requirement: ナビゲーション実行中の画像ロード割り込み防止
全てのナビゲーションメソッド（`jumpToPage`、`scrollToImage`、`scrollToEdge`）の実行中、途中で読み込まれる画像の `load` イベントによる自動レイアウト更新はナビゲーション完了まで抑制されなければならない（SHALL）。これにより、スクロール途中にレイアウトが再計算され、移動先がずれる問題を防止する。

#### Scenario: ナビゲーション中に画像ロードが発火した場合は割り込みしない
- **WHEN** ナビゲーションメソッドが実行されており、移動先へのスクロールが進行中である
- **AND** その他の画像が読み込み完了し `load` イベントが発火する
- **THEN** 自動レイアウト更新は実行されず、ナビゲーション完了後に抑制が解除される
