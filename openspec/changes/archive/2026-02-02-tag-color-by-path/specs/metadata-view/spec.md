## MODIFIED Requirements

### Requirement: Metadata Modal Content
メタデータ閲覧モーダルは、抽出された情報を分かりやすく表示しなければならない（SHALL）。セクションのタイトル（例: "Tags", "Related Works"）は、現在の言語設定に応じた翻訳テキストで表示される。
- **タイトル**: 作品のメインタイトルをモーダルの上部に強調表示する。
- **タグ**: クリック可能なチップ形式で表示し、クリックすると別タブでリンク先を開く。**タグは種別（artist, character 等）に応じて色分けして表示する。**
- **関連作品**: サムネイルとタイトルを表示し、クリックすると別タブでリンク先を開く。
- **バージョン情報**: 現在のアプリケーションバージョンとリリースタイプ（Stable/Unstable）を、識別しやすい形式で表示する。

#### Scenario: バージョン情報の表示
- **WHEN** ユーザーが作品情報モーダルを開く
- **THEN** モーダル内に現在のバージョン（例: `v1.2.0`）が表示される
- **AND** 開発版（unstable ブランチ由来）の場合、`unstable` であることが明示される（例: `v1.2.0-unstable`）

#### Scenario: 翻訳されたセクションラベルの表示
- **WHEN** 日本語環境で作品情報モーダルが表示されている
- **THEN** 「タグ」や「関連作品」といった日本語のセクションタイトルが表示される

#### Scenario: タグの色分け表示
- **WHEN** ユーザーがメタデータモーダルを開く
- **THEN** 各タグはその種別に応じた背景色で表示される
- **AND** 種別が不明なタグはデフォルトのグレー背景で表示される
