## ADDED Requirements

### Requirement: キーボード・ショートカットによるランダム遷移
システムは、キーボード・ショートカットによっても「おすすめ（ランダム）」機能を発火できなければならない (SHALL)。発火後の遷移ロジック（関連作品リストから `isPrivate` でないものの中からランダム選択・遷移）は既存のボタン発火と同一でなければならない。

#### Scenario: キーボード・ショートカットによる遷移
- **WHEN** ユーザーが指定された右手側キーを押下する
- **AND** 関連作品リストに `isPrivate` でない作品が存在する
- **THEN** システムは既存と同じランダム選択ロジックで1つの作品を選び、その `href` へ `window.location.href` を更新して遷移する

#### Scenario: キーボード・ショートカットが HelpModal で表示される
- **WHEN** ユーザーが HelpModal を開く
- **THEN** ランダムジャンプのショートカットキーがラベルと説明文と共に表示される
