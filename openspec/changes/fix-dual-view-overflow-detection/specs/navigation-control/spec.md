## ADDED Requirements

### Requirement: 見開き表示での最終ページ超過時の Info 画面自動表示
見開き表示が有効で、現在の閲覧位置が最終ページペアであるとき、「次へ」方向のナビゲーション操作で info 画面（メタデータモーダル）が正しく表示されなければならない（SHALL）。オーバーフロー判定は、見開き調整による実際の移動先インデックスを考慮した後に行われなければならない。

#### Scenario: 見開き表示で最終ページペアの「次へ」を操作した場合
- **WHEN** 見開き表示が有効で、最終ページペア（例: インデックス N-2, N-1）を閲覧している
- **AND** ユーザーが「次へ」キーまたは画像クリックによる「次へ」操作を行う
- **THEN** メタデータモーダル（info 画面）が自動で表示される

#### Scenario: 見開き表示で最終ページペアの「次へ」でモーダルが既に表示されている場合
- **WHEN** 見開き表示が有効で、最終ページペアを閲覧しており、メタデータモーダルが既に表示されている
- **AND** ユーザーが「次へ」操作を行う
- **THEN** モーダルの状態は変わらず、二重表示にならない
